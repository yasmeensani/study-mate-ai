# 📚 Study Mate AI - Intelligent Study Tutor

Study Mate AI is an AI-powered educational assistant built using **Google Gemini API** and **Gradio**. It helps students understand concepts, solve problems, learn programming, and get step-by-step explanations in a simple and interactive way.

## ✨ Features

- 🤖 AI-powered tutoring using Google Gemini
- 📖 Answers study-related questions
- 🧠 Explains concepts in simple language
- 📝 Provides step-by-step solutions
- 💻 Helps with programming questions
- 🔢 Shows calculation steps for numerical problems
- 📌 Uses headings, bullet points, and examples for better learning
- 🌐 Interactive web interface using Gradio


## 🛠️ Technologies Used

- Python
- Google Gemini API
- Google GenAI SDK
- Gradio
